# Telegram Антиспам Бот

Этот проект представляет собой Telegram-бота для автоматической модерации чатов и удаления спама. Бот использует технологии машинного обучения для классификации сообщений на "спам" и "не спам" (ham).

## Основные возможности

- **Автоматическая фильтрация спама:** Бот анализирует входящие сообщения с помощью предобученной ML-модели на базе логистической регрессии и удаляет потенциальный спам.
- **Блокировка нежелательных репостов:** Автоматически удаляет пересланные сообщения от ботов и из посторонних каналов (разрешены репосты только из "своего" канала).
- **Журнал действий:** Бот отправляет уведомления обо всех удаленных сообщениях в специальную группу-журнал.
- **Управление через интерфейс:** В группе-журнале администраторы могут:
  - Выдать бан пользователю (кнопка "Бан").
  - Обработать ложное срабатывание (кнопка "Ложное"). При этом сообщение добавляется в класс "не спам" (ham) и модель автоматически дообучается в фоновом режиме.
- **Ручное пополнение датасета:** Администраторы могут использовать команду `/add <текст>` в группе-журнале, чтобы вручную разметить текст как спам или хам с последующим переобучением модели.

## Технологии

Разработка ведётся на языке Python.
- **Бот-фреймворк:** [aiogram 3.x](https://docs.aiogram.dev/)
- **Машинное обучение:** `scikit-learn`, `pandas`, `joblib`.
- **Обработка текста:** `nltk`, `pymorphy3`.

## Установка и запуск

1. **Клонируйте репозиторий и установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Настройка окружения:**
   Создайте файл `.env` в корневой директории:
   ```env
   TOKEN="Ваш_Токен_Бота"
   JOURNAL_CHAT_ID="-100..."    # Группа для логов
   CHAT_IDS="-100..."           # Модерируемые чаты
   CHANNEL_ID="-100..."         # Свой канал (для разрешенных репостов)
   ```

3. **Подготовка модели:**
   Если у вас уже есть начальные данные в `data/data.txt`, обучите модель:
   ```bash
   python src/classificator.py
   ```

4. **Запуск бота:**
   ```bash
   python main.py
   ```

Бот автоматически переобучается при добавлении новых данных администраторами через интерфейс Telegram.
